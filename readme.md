Run Project
Step 1: 
```npm install```

Step 2:
```
cd backend

node seed.js  // to create admin and user default in database also helps to add business details
```

Step 3:
```
cd backend
npm run start // which runs the server and helps to connect with database
```

Step 4:
```
cd ..
cd frontend
npm run dev // run frontend of the code and connect with backend
```
